考勤管理系统1.0.0版本

程序结构：

程序分为两部分：监测部分，查询部分。
监测部分通过持续运行的main方法实现，不断将数据库中的MAC地址数据与当前连接WiFi的MAC地址数据做对比。
找出异常数据将其isabsence属性设为1来标识其缺勤。
查询部分使用JSP，将代码托管到tomcat上，老师和学生皆可登录（身份自动识别），查询自己权限范围内的信息。
教师拥有一定的修改信息的权限。

使用说明：

将设备接入目标WiFi。
点入pra.backstage.listener.Listen，执行。后台可自动检测当时时间，从数据取出相应数据进行对比。
jsp代码运行即可。

注意：

tomcat中若没有SQL的相关JAR包，运行会失败，导入即可。
